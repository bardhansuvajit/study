Different from the actual DOM its lightweight and in memeory representation of our component tree where each node represents each component & its properties. When the state/ data of a component changes, React updates the node in vDOM to reflect new state Then it compares the current version with previous version of vDOM to identify the nodes that should be updated. It will then update those nodes in the actual DOM Technically React doesnt update the DOM itself, its done by companion library called React-DOM REACT || REACT-DOM in package.json there are these 2 dependencies react-dom helps React interact with the browser DOM.
