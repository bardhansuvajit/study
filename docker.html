DOCKER with Laravel DEFINATION Docker lets u build and run projects in containers. It's a platform for developing, shipping and running your application. Typically when u build/ host a laravel project, u need minimum these 3 components installed - a server, a adatabase and php. With docker u can build a container for each component Once u createDocker build for your project, u can use it to rebuild your project on any other computer without spending any time configuring your server manually INSTALL Download docker + WSL2 check in terminal `docker -v` Settings to check > Enable WSL2 based engine + Enable Docker compose v2
